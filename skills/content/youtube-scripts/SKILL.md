# YouTube Scripts — Skrypty video z wysoką retencją

Pisze skrypty na YouTube optymalizowane pod retencję widzów. Hook writing, struktura 3-aktowa, pattern interrupts, CTA placement.

## Kiedy używać

- Pisanie skryptu na video YouTube (tutorial, vlog, thought leadership)
- Optymalizacja tytułu i thumbnail
- Poprawa retencji istniejących filmów
- Planowanie serii video
- Użytkownik mówi: "napisz skrypt", "video na YouTube", "jak zacząć film"

## Zależności

- **Opcjonalne:** `voice-profile` → ton i styl autora
- **Opcjonalne:** `copywriting` → formuły (PAS, AIDA) do intro
- **Opcjonalne:** `content-repurpose` → video → inne formaty
- **Opcjonalne:** `content-strategy` → topic clusters

## Struktura skryptu: 3-aktowy model

```
AKT 1: HOOK + SETUP (0-30s) — 15% czasu
├── Hook (0-5s): zatrzymaj scrollowanie
├── Kontekst (5-15s): dlaczego to ważne
└── Promise (15-30s): co widz dostanie

AKT 2: WARTOŚĆ (30s - 80% filmu) — 70% czasu
├── Punkt 1 + przykład
├── Pattern interrupt
├── Punkt 2 + przykład
├── Pattern interrupt
├── Punkt 3 + przykład
└── (opcjonalnie: Punkt 4-5)

AKT 3: ZAMKNIĘCIE (ostatnie 15%) — 15% czasu
├── Podsumowanie (key takeaway)
├── CTA (subskrybuj / obejrzyj kolejne video)
└── Outro
```

## Hook — pierwsze 5 sekund

Hook decyduje czy widz zostanie. 80% widzów odpada w pierwszych 30 sekundach.

### Typy hooków:

| Typ | Przykład | Siła |
|-----|---------|------|
| **Kontrowersja** | "Przestań robić landing page. Poważnie." | ★★★★★ |
| **Zaskakujący fakt** | "93% startupów nie mierzy retention." | ★★★★★ |
| **Pytanie** | "Ile czasu marnujesz na ręczną pracę?" | ★★★★ |
| **Before/After** | "Rok temu: 0 klientów. Dziś: 47." | ★★★★ |
| **Problem** | "Twoja strona nie konwertuje. Oto dlaczego." | ★★★★ |
| **Preview wyniku** | "Pod koniec tego video będziesz wiedzieć jak..." | ★★★ |
| **Historia** | "Wczoraj straciłem klienta za 12 000 zł miesięcznie." | ★★★★★ |
| **Demonstracja** | [Pokaż efekt końcowy od razu] | ★★★★ |

### Formuła hooka:

```
[HOOK] + [KONTEKST] + [PROMISE]

Przykład:
"93% startupów nie mierzy retention — i potem się dziwią że klienci odchodzą.
Dziś pokażę Ci 3 metryki, które musisz śledzić od pierwszego dnia.
Zostań do końca — dam Ci gotowy dashboard do skopiowania."
```

### Czego NIE robić w hook:
- "Cześć, nazywam się..." → NIE (to intro, nie hook)
- "W dzisiejszym odcinku..." → NIE (nudne)
- "Zanim zaczniemy, zasubskrybuj..." → NIE (za wcześnie)
- Długie logo/intro animation → NIE (ludzie skipują)

## Pattern Interrupts

Co 2-3 minuty coś musi się zmienić, żeby utrzymać uwagę:

| Typ | Przykład | Kiedy |
|-----|---------|-------|
| B-roll | Zmiana kadru, screencast, zdjęcia | Co 15-30s |
| Tekst na ekranie | Key point, statystyka, cytat | Przy ważnych danych |
| Zmiana tempa | Szybki montaż / slow-mo | Przy podsumowaniach |
| Pytanie retoryczne | "Ale co jeśli to nie wystarczy?" | Między sekcjami |
| Sound effect | Woosh, pop, ding | Przy listach, przejściach |
| Zoom in/out | Zbliżenie na twarz | Przy emocjonalnych momentach |
| Humor/anegdota | Krótka historia, meme | Co 3-4 min |

## Template skryptu

```markdown
# [TYTUŁ VIDEO]
# Długość docelowa: [X] min
# Typ: tutorial / thought leadership / case study / vlog

---

## HOOK (0:00 - 0:05)
[Zdanie otwierające — kontrowersja, fakt, pytanie]

## KONTEKST (0:05 - 0:20)
[Dlaczego to ważne — ból widza, skala problemu]

## PROMISE (0:20 - 0:30)
[Co widz dostanie — "Pod koniec będziesz wiedzieć jak..."]
[Opcjonalnie: "Zostań do końca, bo w punkcie 3 pokażę X"]

---

## PUNKT 1: [NAZWA] (0:30 - X:XX)

### Teza
[Główna myśl — 1-2 zdania]

### Przykład / Demonstracja
[Konkretny case, screencast, dane]

### Takeaway
[Co widz powinien zrobić z tą wiedzą]

> Pattern interrupt: [B-roll / tekst / pytanie]

---

## PUNKT 2: [NAZWA] (X:XX - X:XX)
[Powtórz strukturę jak punkt 1]

> Pattern interrupt

---

## PUNKT 3: [NAZWA] (X:XX - X:XX)
[Powtórz strukturę]

---

## PODSUMOWANIE (X:XX - X:XX)
[3 key takeaways — krótko, bullet points]

## CTA (ostatnie 15-30s)
[1. Subskrybuj jeśli chcesz więcej o [temat]]
[2. Obejrzyj to video → [link/card do powiązanego]]
[3. Opcjonalnie: link w opisie do narzędzia/zasobu]

## OUTRO
[Krótkie — max 5s. "Do zobaczenia w następnym." + end screen]
```

## Tytuł i Thumbnail

### Tytuł — formuły:

| Formuła | Przykład |
|---------|---------|
| How to + [wynik] | "Jak zdobyć 100 klientów bez reklam" |
| Liczba + [temat] | "5 błędów, które zabijają Twój SaaS" |
| I [did X] — [wynik] | "Zautomatyzowałem sprzedaż — oto wyniki" |
| Stop [bad thing] | "Przestań robić te 3 rzeczy w marketingu" |
| [Autorytet] + [secret] | "Jak startupy z Y Combinator zdobywają klientów" |
| Why [contrary] | "Dlaczego NIE powinieneś robić SEO" |
| [X] vs [Y] | "Cold email vs LinkedIn outreach — co działa?" |

**Zasady tytułu:**
- Max 60 znaków (reszta ucięta na mobile)
- Keyword na początku (SEO)
- Nie kłam (clickbait = niska retencja = algorytm karze)
- Testuj 2-3 warianty (YouTube pozwala zmienić tytuł)

### Thumbnail:

```
ZASADY:
- 1280x720px (16:9)
- Twarz z emocją (zaskoczenie, szok) = +30% CTR
- Max 3-5 słów tekstu (duży, czytelny)
- Kontrastowe kolory (żółty/czerwony na ciemnym tle)
- NIE powtarzaj tytułu — uzupełnij go
- Spójny styl per kanał (rozpoznawalność)
```

## Typy video SaaS

| Typ | Długość | Retencja target | Przykład |
|-----|---------|----------------|---------|
| Tutorial | 8-15 min | 40-50% | "Jak skonfigurować X w 10 min" |
| Thought leadership | 5-10 min | 35-45% | "Dlaczego SEO nie działa dla SaaS" |
| Case study | 8-12 min | 45-55% | "Jak firma X osiągnęła 100K MRR" |
| Comparison | 6-10 min | 40-50% | "Narzędzie A vs B — co wybrać?" |
| Behind the scenes | 5-8 min | 30-40% | "Dzień z życia foundera SaaS" |
| Shorts (< 60s) | 30-60s | 70-90% | Jeden tip, hook → payoff |

## Metryki video

| Metryka | Target | Co oznacza |
|---------|--------|-----------|
| CTR (Click-Through Rate) | > 5% | Tytuł + thumbnail działają |
| Avg. view duration | > 50% długości | Retencja OK |
| Retention at 30s | > 70% | Hook działa |
| Likes/views ratio | > 4% | Treść wartościowa |
| Comments | > 1% views | Engagement |
| Subscriber gain | > 2% views | Widz chce więcej |

## Pułapki

1. **Za długie intro** → hook w 5s, nie w 30s
2. **Brak pattern interrupts** → monotonia = odpływ widzów po 2 min
3. **CTA na początku** → "zasubskrybuj" w pierwszych 30s = irytacja
4. **Perfekcjonizm** → lepiej opublikować 80% perfect niż nigdy nie opublikować
5. **Brak structury** → "gadanie do kamery" bez planu = niska retencja
6. **Kopiowanie formatu z podcastu** → YouTube ≠ podcast, potrzebuje wizuali
